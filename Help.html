Gathology Description

Gathology is a generic platform, you can create an application-like form which can be reached from android, web and soon will be avialable on iphone.
Purpose:

The purpose of Gathology platform is to help scientists and researchers with no software development background to collect the data they need, to build their knowledge base. It also helps scientists to visualize the collected data and visualize geo-locations where data was collected from.

Create a form

- While creating a form you have several elements:

1. Note: This is a static text, you can use to clarify something to the user.
2. Textbox: a field for entring free data.
3. DropDownList: To allow user to choose from several options. This also has a label and id.
-Options' type will be considered as text not numeric, so you won't be able to perform arithmetic operations on the collected choices.
4. Radiogroup: The only difference between Radiogroup and DropDownList is that Radiogroup can be required so users will have to choose an option by themselves
5. Checkbox: To allow user to determine if something is true or false.
6. Image: This is a field for user to upload image, and you will get the link in results.
7. Video: This is a field for user to upload vedio, and you will get the link in results.
8. Document: This is a field for user to upload text files and documents, and you will get the link in results.

Properities: - ID: the id that specifies this element in showing the results later. It has to be unique in form.
- Label: This will be the hint or the prompt message.
- Maximum Length: The maximum number of characters can be written in a textbox.
- Required: This indicates if user can skip this element or not.
- Multiline: Can the user enter several lines? or just write his data in only one line.
- Input type: Letters only, and this can be used in names and fields that can not contain numbers or other characters.
Numbers only, when you need this field to be numeric to perform arithmetic operations on it. the other option refers to that the text box can accept anything.

Scatter Chart

- You have two options:

1. One column: Visualize the relation between values in a column on X-axis and the number of occurrences of that values on Y-axis

2. Many columns: Visualize the relation between one column on X-axis and many columns on several Y-axes after applying aggregation function on them to prevent repeated values on x-axis. Y-axis columns differ from each other in the color of the dots.

Stacked Column Chart

- Here you can visualize the relation between two data sets. A column of data and the number of occurrences of distinct values under another column.

- In chart: a column will represent a value in the first data set, and a slice of that column will represent the number of occurrences of a value in the second data set in front of the same value in the first data set.
